# World-of-Dights-add-on
Аддон для мода World of Dights: сюда будут депнуты идеи, которые автор мода не одобрил

## что он должен добавить:
- трит и кубит: квантовые и троичные системы (пока только спазм троичного нерва)
- типо Сетунь это типо там троичная система, ещё там ДССП а там типо всякие тоже команды, кусок стека и типо эффективнее Киберии (типо киберия это двоичный код а Сетунь это троичный)
